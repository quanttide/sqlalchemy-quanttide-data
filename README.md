# sql-client

SQL数据库的Python操作库的封装。

## 特性

- 封装各个SQL数据库的Python操作库，提供基本一致的API，只需更换import语句（以及其它一些小调整）即可更换底层库甚至更换数据库类型（如从MySQL更换为PostgreSQL）
